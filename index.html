MAGISTICK : Multiplayer Ninja Fighting Game using Unity

An indie game built using Unity Game Engine and Mirror networking(Equivalent to Unity HLAPI). It's a PVP multiplayer Game where Players can fight it out in a battle involving Kunai's and Swords. It contains a lobby system where players can queue and an Joystick for controlling the player. It involves an IAP purchasing option which is yet to come and will be released for android in the Play store soon.


3D Male Game Character

I had always wanted to learn how 3D game characters in AAA titles where created. This model is a result of 30 hour course in Udemy which involved modelling in Autodesk Maya, Sculpting in Zbrush, Clothing in Marvelous Designer and Texturing in Substance painter. I learnt so much about 3D modelling and the hair was created using XGen in Maya. I rigged it and brought it into a 3D world which I had created in Unity (In the Background).
